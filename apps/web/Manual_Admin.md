# Manual de Administrador (God Mode)
**PETCARE AI PLATFORM**

Este panel está reservado únicamente para los administradores y dueños de la plataforma PetCare AI.

## 1. Acceso
El acceso a este panel no está disponible públicamente en la interfaz. Debes iniciar sesión con una cuenta que tenga el rol `ADMIN` a nivel de base de datos.
Una vez autenticado, serás redirigido automáticamente al **Admin Dashboard**.

## 2. Visión Global (Métricas)
El panel te permite tener una visión a vuelo de pájaro de la plataforma:
- **Usuarios Totales:** Monitorea el crecimiento de la base de datos (clínicas y dueños registrados).
- **Consultas IA:** Mide el volumen de uso de la API de Inteligencia Artificial para predecir costos de servidor.
- **Ingresos Estimados (MRR):** Proyección financiera mensual basada en las suscripciones activas de clínicas.

## 3. Moderación
- En futuras actualizaciones, este panel te permitirá moderar foros de la comunidad y suspender cuentas que incumplan los términos de servicio.
